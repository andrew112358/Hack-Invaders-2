# CapyCatch

A Pokémon-style browser game — explore the meadow, battle wild capybaras, and catch all six elemental types!

## Play

```bash
cd Hack-Invaders-2
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow keys | Walk |
| Space / Enter | Advance dialog |
| C | Open collection |

## Gameplay

- You start with **Capybuddy** (Normal).
- Walk on **grass** or **flower patches** for random encounters.
- In battle: **Fight**, **Catch** (Capy Balls — weaken them first for better odds), **Bag**, or **Run**.
- Catch all six: Capybuddy, Mossbara, Emberbara, Voltbara, Aquabara, Zenbara.
